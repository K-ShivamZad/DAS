## 📂 Repository Structure & Modules

This repository is structured sequentially to cover the complete Data Science lifecycle:

### 1. Data Manipulation & Visualization
* **`python_refresher.ipynb`**: Core Python programming, data structures, and algorithms.
* **`Data_Visualization.ipynb`**: Exploratory Data Analysis (EDA) using Matplotlib and Seaborn, plus Web Scraping with BeautifulSoup.

### 2. Database Management
* **`SQL_Mastery.ipynb`**: Relational database design, complex JOINs, GROUP BY aggregations, and subqueries using SQLite and Pandas.

### 3. Machine Learning & Deep Learning
* **`Machine_Learning_Mastery.ipynb`**: End-to-end Scikit-Learn pipelines, feature engineering, and model persistence (Joblib). Includes a House Price Prediction model.
* **Deep Learning**: Introduction to Neural Networks and building multi-layer perceptrons with TensorFlow and Keras on the MNIST dataset.

### 4. Applied AI & Natural Language Processing
* **`Project_3_RAG_Assistant.ipynb`**: A state-of-the-art Retrieval-Augmented Generation (RAG) pipeline. 
  * Transcribes lecture audio using **OpenAI's Whisper**.
  * Chunks text using LangChain.
  * Creates vector embeddings using `SentenceTransformers`.
  * Performs semantic search using **FAISS** to act as an AI Teaching Assistant.

### 5. Deployment
* **`Web_Deployment.ipynb` & `app.py`**: Wrapping Machine Learning models into interactive web applications and RESTful APIs using the **Flask** framework.
